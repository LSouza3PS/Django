Bem-vindo ao Sistema de Gestão de Tarefas

Este é um projeto fictício desenvolvido por Leandro Benedito de Souza para demonstrar suas habilidades em desenvolvimento web usando o framework Django. O Sistema de Gestão de Tarefas oferece uma plataforma intuitiva e eficiente para organizar suas atividades diárias.

Com este sistema, você pode:

  • Criar tarefas com descrição, data de conclusão e prioridade
  • Editar e excluir tarefas facilmente
  • Marcar tarefas como concluídas
  • Visualizar suas tarefas de maneira organizada e filtrada
